## COMMON ISSUES & FIXES

| Issue | Solution |
|-------|----------|
| `ModuleNotFoundError: No module named 'listings'` | Run: `python manage.py migrate` |
| Emails not sending | Check `.env` EMAIL settings |
| Migration errors | Delete `db.sqlite3` and rerun `migrate` |
| Verification code not working | Check code matches exactly (case sensitive) |
| Form not submitting on mobile | Already fixed! Clear cache and retry |
| Can't access dashboard after verify | Clear browser cookies, login again |
